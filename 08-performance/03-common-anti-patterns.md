# Common Anti-Patterns

## Overview

Certain SQL practices can lead to poor performance or incorrect results. These are known as anti-patterns.

## Examples

- Using `SELECT *` instead of specifying columns.
- Not using indexes on frequently searched columns.
- Using subqueries when a JOIN would be better.
- Ignoring NULL handling in conditions.
